# Discovery admin

**Discovery** in the admin sidebar is a **superadmin** global view of discovered devices and networks across the platform (aggregates beyond a single site page).

## When to use it

- Audit what collectors have found fleet-wide
- Find orphaned or unexpected networks
- Cross-check site discovery health

Day-to-day credential and Meraki work still belongs under each site’s [Site discovery](site-discovery.md) page.
